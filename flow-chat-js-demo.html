Flow Chat JS

A Javascript plugin to help developers build simple conversational chat flow.

This plugin allows you to input a JSON file and build a conversational chat flow. Each option the user select leads them to the next message. Each user can have their own journey.

Some of the usecases for this are:

  • - Storytelling
  • - FAQs and Support
  • - Games
  • - Teaching

Find the complete documentation for usage at FlowChat JS on Github.
